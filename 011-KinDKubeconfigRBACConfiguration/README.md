# Project 011: Create Read Only Kubeconfig File

## Goal

In this lab, you will learn how to generate a kubeconfig file with read-only access.

## Prerequisites

- Ubuntu 20.04 OS (Minimum 2 core CPU/8GB RAM/30GB Disk)
- Docker
- Docker Compose

## Vagrant home

- Windows

`c:\devbox`

- Mac

`~/devbox`
